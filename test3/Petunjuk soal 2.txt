2. Buat fungsi dengan php untuk menjadikan array tersebut 1 dimensi dan urut
Contoh: 
array('1' => array('ID' => '1',
      		       'DETAIL' => 'DEPARTMENT 1',
      		       'SEQUENCE' => '-'
      '3' => array('ID' => '3',
				   'DETAIL' => 'DEPARTMENT 3',
		           'SEQUENCE' => '--'
      '13' => array('ID' => '13',
		       		'DETAIL' => 'DEPARTMENT 13',
		            'SEQUENCE' => '---'
      '2' => array('ID' => '2',
				   'DETAIL' => 'DEPARTMENT 2',
      		       'SEQUENCE' => '-'
      '8' => array('ID' => '8',
		       	   'DETAIL' => 'DEPARTMENT 8',
		           'SEQUENCE' => '--'));


NOTE: - SEQUENCE MENUNJUKKAN TINGKATAN ARRAY (ARRAY TINGKAT 1 = '-', ARRAY TINGKAT 2 = '--', DST)
      - DATA DENGAN PARENT_ID KOSONG DIANGGAP SEBAGAI ARRAY TINGKAT 1
      - PARENT_ID DIGUNAKAN UNTUK MENCARI PARENT DATA TERSEBUT
